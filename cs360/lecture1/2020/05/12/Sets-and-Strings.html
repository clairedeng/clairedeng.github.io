If an object $x$ is an element of a set $A$, we write $ x \in A$. Otherwise, we write $x \notin A$.

If $A$ and $B$ are sets, we say $A$ is a subset of $B$ if all the elements of $A$ are also elements of $B$, denoted as $A\subseteq B$. In this situation, $B$ is a superset of $A$. $A\subseteq A$ for all sets $A$. $\phi \subseteq A $ for all sets A.

If $A \subseteq B$ but $A\neq B$, we say $A$ is a proper subset of $B$, denoted as $A \subsetneq B$.

The intersection of set $A$ and $B$ is denoted by $A\cap B$, the set which contains all elements $A$ and $B$ have in common.

The union of set $A$ and $B$ is denoted by $A\cup B$, the set which contains all objects that are elements of either A or B.

Infinite Sets

The set of all natural numbers is a valid infinite set.

A set $A$ is finite if $|A| \in N$, and otherwise we say $A$ is infinite.

The cardinality of two infinite sets and is the same, $|A|=|B|$, even though $A \subsetneq B$. The reason is that $A$ and $B$ can be put in one-to-one correspondence: for each element $x$ in $A$, we can find a match for it in $B$, which will be $x-1$.

Not all pairs of infinite sets can be put in one-to-one correspondence with each other: there are different cardinalities of infinite sets.

The smallest infinite cardinality is the cardinality of the natural numbers.

A set whose cardinality is equal to or smaller than that of $N$ is called countable.

A set which is not countable is called uncountable. eg. the set of real numbers $R$

Strings

An alphabet is a nonempty finite set, denoted by $\Sigma$. It is the set of symbols that a program needs to recognize.

Elements of an alphabet $\Sigma$ are called characters or symbols.

A string is a finite sequence of elements of $\Sigma$.

The length of a string $x$, denoted $|x|$, is the number of characters in the sequence $x$.

Strings are not allowed to be infinite.

The string which contains zero characters is called the empty string, denoted by $\epsilon $. The length of $\epsilon$ is 0. $\epsilon$ is a string over every alphabet $\Sigma$.

If $x$ and $y$ are two strings over the same alphabet $\Sigma$, the concatenation of $x$ and $y$ is the string $xy$ consisting of the sequence of characters in $x$ followed by the sequence of characters in $y$. The concatenation of $\epsilon$ with any string $x$ equals $x$: we have $\epsilon x = x\epsilon = x$. The length $|xy|$ of the concatenation of $x$ and $y$ equals $|x|+|y|$.

Given an alphabet $\Sigma$, a language over $\Sigma$ is a set of strings over $\Sigma$.

A language can only contain finite strings, but the language itself may be infinite.

$A^* = A^0\cup A^1\cup A^2\cup…$ is called Kleene star. It is the union of $A^n$ over all $n \in N$ and it’s an infinite set.

If $\Sigma$ is an alphabet, then $\Sigma^{*}$ is the set of all strings over that alphabet, since every string over $\Sigma$ can be formed by concatenating characters of $\Sigma$ together. Any language $A$ over $\Sigma$ must be a subset of $\Sigma^{*}$. The set $\Sigma^{*}$ is countable. Since all language over $\Sigma$ are subsets of $\Sigma^{*}$, all languages are countable.
