Sets

A set is a collection of distinct elements. eg. $A = {2,3,5}$.

The size or cardinality of a set is the number of elements it contains, denoted as $$ A $$. eg. $ A = 3$

The elements in a set are unordered and must all be distinct. eg. ${3} \ne 3, {{3}} \ne {3}$

The empty set is a set with no elements, denoted as $\phi$ or ${}$. The cardinality of the empty set is 0: $\phi = 0$.

Question 1: What is $ {\phi} $?
$ {\phi} = {{}} = 1$. It is the set containing the empty set.

If an object $x$ is an element of a set $A$, we write $ x \in A$. Otherwise, we write $x \notin A$.

If $A$ and $B$ are sets, we say $A$ is a subset of $B$ if all the elements of $A$ are also elements of $B$, denoted as $A\subseteq B$. In this situation, $B$ is a superset of $A$. $A\subseteq A$ for all sets $A$. $\phi \subseteq A $ for all sets A.

If $A \subseteq B$ but $A\neq B$, we say $A$ is a proper subset of $B$, denoted as $A \subsetneq B$.

The intersection of set $A$ and $B$ is denoted by $A\cap B$, the set which contains all elements $A$ and $B$ have in common.

The union of set $A$ and $B$ is denoted by $A\cup B$, the set which contains all objects that are elements of either A or B.

For all sets $A$ and $B$, we have $A\cap B \subseteq A$ and $A\subseteq A\cup B$, $A\cap \phi = \phi$ and $A\cup \phi = A$.

Question 2: What is $\phi \cup {\phi} \cup {{\phi}}$?

$\phi \cup {\phi} \cup {{\phi}} = {\phi, {\phi}}$

The set subtraction of set $A$ and $B$ is denoted by $A \setminus B$, the set of all elements which are in $A$ but not in $B$. $ A\setminus B $ does not have to equal $ A - B $. eg. if $A = {2,3,5}$ and $B = {1,2,3}$, then $A\setminus B = {5}$ and $ A\setminus B = 1\neq A - B = 3-3=0$.

The complement of set $A$ is denoted by $\overline{A}$, the set of all things not in $A$. $\overline{A}$ is only defined with respect to some universe set $U$ which is a superset of $A$. This way, $\overline{A}$ will be equal to $U\setminus A$.

If $A$ is a set, then the set of all subsets of $A$ is denoted by $P(A)$ or $2^A$, the power set of $A$. eg. $P({2,3,5}) = {\phi, {2}, {3}, {5}, {2,3}, {2, 5}, {3, 5}, {2,3,5}}$. $P(\phi) = {\phi}$. If $A$ is a finite set, then the size of $P(A)$ will always be exactly $2^{ A }$.
