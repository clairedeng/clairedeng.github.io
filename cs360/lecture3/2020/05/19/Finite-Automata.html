Deterministic finite automata(DFA)

The deterministic finite automata is a very simple computational model, in which the computing device (“automaton”) simply has a finite set $Q$ of internal states it can be in. This automaton will read an input string one symbol at a time,and for each symbol it reads, it may move from its current internal state to one of the other internal states according to a fixed set of rules. This automation will have some of its internal states labeled “accepted state”, and the rest labeled “reject state”. Once the automaton finishes reading the string, it will accept the string(effectively outputting “1” or “True”) if the final state it has reached is an accept state, and will reject the string (effectively outputting “0” or “False”) if the final state is a reject state.

A deterministic finite automaton(DFA) is a tuple $(Q, \Sigma, \delta, q_0, F)$ where

  1. $Q$ is a finite nonempty set, representing the set of internal states of the automaton.
  2. $\Sigma$ is a finite nonempty set, representing the alphabet of the input strings the DFA can receive.
  3. $\delta$ is a function $\delta : Q \times \Sigma \rightarrow Q$ called the transition function of the DFA (the interpretation of this function is that if the automaton has internal state $q$ and reads the symbol $a$, it will move to the internal state $\delta(q,a)$)
  4. $q_0$ is an element of $Q$ called the initial state or start state, representing the internal state of the DFA before it reads any input symbols
  5. $F$ is a subset of $Q$, representing the set of accept states of the automaton.

E.g.

Taking a Boolean string as input, accepting it if its parity is 1(that is, if the number of 1 symbols in the string is odd).

$Q = {q_0, q_1}$ where $q_0$ represents “parity has been 0 so far” and $q_1$ represents “parity has been 1 so far”.

since we are going to receive Boolean strings as input.

$\delta(q_0, 0) = q_0, \delta(q_0, 1) = q_1, \delta(q_1, 0) = q_1, \delta(q_1, 1) = q_0$

The start state will be $q_0$.

$F = {q_1}$.

Question 1: What does the DFA in following figure do?

The only way to reach $q_1$ is to see a 1, which means the only way to reach $q_2$ is to see the symbols 10 in a row, and so the only way to reach $q_3$ for the first time is to see the sequence 101. In conclusion, this DFA accepts all strings which do not contain 101 as a substring.

Definition 3.2: Let $M$ be a DFA. The language recognized by $M$, denoted $L(M)$, is the set of all strings in $\Sigma^{*}$ accepted by $M$, where $\Sigma$ is the alphabet of $M$.

Nondeterministic finite automata

An NFA is like a DFA, except that a state q may have multiple outgoing arcs with the same label. An NFA accepts a string x if some path it can follow for x leads it to an accept state. If an NFA is at a state $q$ and there is an $\epsilon$-transition from $q$ to $q’$, then the NFA is allowed to go from q to $q’$, or remain at $q$. NFAs need not always have an allowed transition. If the NFA is at state $q$ and sees a 0(no arc going out of q with the label 0), it must reject the entire string.

An NFA will be a tuple $(Q, \Sigma, \delta, q_0, F)$ where

  1. $Q$ is a finite nonempty set, representing the set of internal states of the automaton.
  2. $\Sigma$ is a finite nonempty set, representing the alphabet of the input strings the NFA can receive.
  3. $\delta$ is a function which takes in a state $q\in Q$ and either a symbol in $\Sigma$ or $\epsilon$, and outputs a set of states to which the NFA may transition.
  4. $q_0$ is an element of $Q$ called the initial state or start state, representing the internal state of the DFA before it reads any input symbols
  5. $F$ is a subset of $Q$, representing the set of accept states of the automaton.

E.g.

since we are going to receive Boolean strings as input.

$\delta(q_0, 0) = {q_0}, \delta(q_0, 1) = {q_1, q_3}, \delta(q_0, \epsilon) = \phi$

$\delta(q_1, 0) = {q_2}, \delta(q_1, 1) = \phi, \delta(q_1, \epsilon) = \phi$

$\delta(q_2, 0) = \phi, \delta(q_2, 1) = \phi, \delta(q_2, \epsilon) = q_1$

$\delta(q_3, 0) = \phi, \delta(q_3, 1) = \phi, \delta(q_3, \epsilon) = \phi$

The start state will be $q_0$.
