Variable Selection in Least Square Regression

05 Jun 2020 » Leetcode, STAT444

Two Problems with the least square estimate $X\hat\beta = HY$:

  • Prediction accuracy: LS has low bias, but if n is not much greater than p, then they have large variance. Multicollinearity also causes variance inflation in LS regression.
  • Model interpretability: Including irrelevant variables leads into unnecessary complexity in the model. One needs to sacrifice some small details to get the “big picture”.

Best-subset selection

Consider the model $Y = f(X_1, …, X_p) + \epsilon$. The subset selection is the process of selecting $q < p$ explanatory variates in modelling the function f.

To fit a separate least squares regression for each possible combination of the $p$ predictors, there is a total of ${p \choose 0} + {p\choose 1} + … + {p\choose p} = 2^p$ models to be fitted.

When p greater than 30 to 40, we are unable to use the algorithm to compute the result.

  1. Let $M_0$ denote the null model, which contains no predictor. This model simply predicts the sample mean for each observation.

  2. For $k = 1, …, p$:

a) Fit all $p\choose k$ models that contain exactly $k$ predictors.

b) Pick the best among these $p\choose k$ models, and call it $M_k$. Here best is identified as having the smallest RRS, or equivalently largest $R^2$.

  1. Select a single best model from among $M_0, …, M_p$ using cross-validated prediction error, Mallow’s $C_p$(AIC), BIC, or adjusted $R^2$.

Forward-stepwise selection

  1. Let $M_0$ denote the null model, which contains no predictors. This model simply predicts the sample mean for each observation.

  2. For $k = 0, …, p-1$:

a) Consider all p-k models that augment the predictors in $M_k$ with $k$ additional predictor.

b) Choose the best among these p-k models, and call it $M_{k+1}$. Here best is identified as having the smallest RSS, or equivalently largest $R^2$.

  1. Select a single best model from among $M_0, …, M_p$ using cross-validated prediction error, Mallow’s $C_p$(AIC), BIC, or adjusted $R^2$.

Backward-stepwise selection

  1. Let $M_0$ denote the null model, which contains all p predictors.

  2. For $k = p, p-1, …, 1$:

    a) Consider all k models that contain all but one of the predictors in $M_k$, for a total of $k-1$ additional predictors.

    b) Choose the best among these $k$ models, and call it $M_{k-1}$. Here best is identified as having the smallest RSS, or equivalently largest $R^2$.

  3. Select a single best model from among $M_0, …, M_p$ using cross-validated prediction error, Mallow’s $C_p$(AIC), BIC, or adjusted $R^2$.

A Hybrid Approach

In the hybrid method, variables are added to the model sequentially, in analogy to forward selection. However, after adding each new variable, the method may also remove any variables that no longer provide an improvement in the model fit.

The hybrid approach attempts to more closely mimic best subset selection while retaining the computational advantages of forward and backward stepwise selection.

The stepwise methods are all greedy algorithms, producing a nested sequence of models. However, the stepwise methods are computationally cheaper than best subset selection.

Stepwise methods are more constraint than the best subset selection, hence they will have lower variance, but perhaps more bias.

Shrinkage methods: an introduction to regularization

All variable selection methods explained above are within the LS framework.

What if multicollinearity exists or what if $n < p$?

Shrinkage/regularization methods useful under these circumstances, and they are computationally more efficient.

  • LS: $\min_\beta(RSS)$
  • Shrinkage: $\min_\beta(RSS) + \lambda\times Pen(\beta)$

The added penalty in the shrinkage estimator, shrinks the vector $\beta$ towards $0$.

Shrinkage methods: Ridge, Lasso, and Elastic Net regression.
