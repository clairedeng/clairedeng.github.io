The Myhill-Nerode Theorem

07 Jun 2020 » Leetcode, CS360

Let $A\sube \Sigma^$ be a language over $\Sigma$, and let $x,y \in \Sigma^$ be two strings (not necessarily strings in A). We say that a string $z\in \Sigma^*$ is a distinguishing extension for $x$ and $y$ if one of $xz$ and $yz$ is in A and the other is not.

two strings $x,y\in \Sigma^*$ are equivalent with respect to A, denoted $x\equiv_A y$, if there is no distinguishing extension between $x$ and $y$.

Equivalence relation:

$x \equiv_A x$

if $x\equiv_a y$ then $y\equiv_a x$.

If $x\equiv_a y$ and $y\equiv_a w$, then $x\equiv_a w$.

For any $x\in \Sigma^$, the equivalence class of x with respect to $A$ will be the set of strings $C\sube \Sigma^$ defined by $C = {y\in \Sigma^*: x\equiv_A y}$.

The equivalence classes with respect to a language $A$ have the following properties:

  • all the equivalence classes are nonempty.
  • any two equivalence classes with respect to A are either exactly equal or else disjoint.
  • The union of all the equivalence classes with respect to A is $\Sigma^*$
  • For any equivalence class C with respect to A, we have either $C\sube A$ or else $C\cap A = \O$.
  • For any two equivalence classes $C$ and $D$ with respect to $A$, if there are strings $x\in C$ and $z\in \Sigma^*$ with $xz\in D$, then all strings $y\in C$ also have $yz \in D$.

Myhill-Nerode Theorem: Let $\Sigma$ be an alphabet and let $A\in \Sigma^*$ be a language over $\Sigma$. Then A is regular if and only if there are finitely many equivalence classes with respect to A. Moreover, if A is regular, then the number of equivalence classes with respect to A is exactly the minimum number of states in a DFA for A.
