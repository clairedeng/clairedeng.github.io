Context-free grammars and languages

06 Jun 2020 » Leetcode, CS360

Definition 1: A context-free grammar (CFG) is a 4-tuple where $V$ is a finite and non-empty set (whose element we will call variables), $\Sigma$ is an alphabet (disjoint from $V$), $R$ is a finite and nonempty set of rules, each of which takes the form for some choice of $A\in V$ and $w\in (V\cup \Sigma)^*$, and $S\in V$ is a variable called the start variable.

Definition 2: Let $G=(V,\Sigma, R, S)$ be a context-free grammar. The yields relation defined by G is a relation defined for pairs of strings over the alphabet $V\cup \Sigma$ as follows: for every choice of strings $u,w,v \in (V\cup \Sigma)^*$ and a variable $A\in V$, provided that the rule $A\rightarrow w$ is included in $R$.

Definition 3: Let $G=(V,\Sigma, R, S)$ be a context-free grammar. For any two strings $x, y\in (V\cup \Sigma)^$ one has that if there exists a positive integer $m$ and strings $z_1, …, z_m \in (V\cup \Sigma)^$ such that

  1. $x=z_1$
  2. $y=z_m$
  3. $z_k\rightarrow_G z_{k+1}$ for every $k\in{1, …, m-1}$.
